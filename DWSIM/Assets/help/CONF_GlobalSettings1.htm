Local Calculation Engine Settings


CPU Parallel Processing
This option enables calculations in multiple processors in parallel. Select maximum numbers of processors for calculation.

CPU SIMD Acceleration
This option enables vector operations through CPU SIMD features.

OpenCL/CUDA Processing
This option enables parallel calculations on Graphic Card (GPU). Select device where calculations are executed.
