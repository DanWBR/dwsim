Master Property Table

This table object is used to display multiple properties of several selected simulation objects of a common type in a common table within the process flowsheet.

After inserting a Master Property Table to the flowsheet, double-click on it to edit.

Parameters



1. Object Type
Here you need to select the type of object you want to select. For instance "MaterialStream" will list all material streams of the simulation for selection.

2. Objects to show
After selecting the object type all objects of that type are listed in this section. You may select the objects to be included in this table.


3. Objects ordering
This option defines the ordering of objects within the table.

4. Properties to Show
This section displays all properties of the selected object type which may be displayed in the table. You can select the properties which are to be displayed in the table.
