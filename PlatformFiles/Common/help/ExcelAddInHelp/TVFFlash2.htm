TVFFlash2

Calculates a Temperature / Vapour Fraction Flash using the selected Property Package.
This second type of TVF Flash allows to define an initial estimation of resulting pressure for faster and more reliable calculations.
For a more detailed explanation of this function got to function TVFFlash

Input Parameters

  • Proppack: - select from model list returned by function GetPropPackList
  • Flashalg: Flash algorithm to be used for calculation:
    0 or 2 = Nested Loops VLE
    1 = Inside-Out VLE
    3 = Inside-Out VLLE
    4 = Gibbs VLE
    5 = Gibbs VLLE
    6 = Nested-Loops VLLE
    7 = Nested-Loops SLE
    8 = Nested-Loops Immisc.
    9 = Simple LLE
  • Pressure: Pressure in [Pa]
  • Enthalpy: Specific enthalpy of mixture in [KJ/Kg]
  • Compounds: A list of component names in the mixture as a one dimensional matrix.
    This is usually defined as a reference to a cell range like "A1:A5" where you list the component names. Available component names are returned by function GetCompoundList
  • Molefractions: A list of component mole fractions in the global mixture as a one dimensional matrix.
    This is usually defined as a reference to a cell range like "B1:B5" where you list the mixture composition.
    The sum of all fractions always has to be 1.
  • IP1 ... IP8: Interaction parameter sets #1 ... #8. This allows to override standard parameters for equilibrium calculation.
  • InitialEstimate: Initial estimate for pressure search, in [Pa].
