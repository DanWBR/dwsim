Joback Group Definition

The Joback group contribution method is used in DWSIM to estimate 9 important properties of pure components. Estimation of properties should be used if no measured data are available. If experimental or literature data are available they can be put into the corresponding field. Some parameters like normal boiling point and critical point are used to estimate other parameters. Therefore it is highliy recommendet to supply experimental data if available!



Joback definition table

  1. ID
    Group ID number
     
  2. Group type
    Joback groups are assigned to a dedicated group type. Groups with the same name but in from different group types will give different results in property estimation.
     
  3. Group Name
    Name of Jobakc group
     
  4. Count UNIFAC
    This row contains the numer of Joback groups contained in the UNIFAC groups as subgoups. This column is calculated automatically and can't be edited.
     
  5. Count Jobakc
    This column allows to put in the number of Joback groups. If it is not possible to define the component completely by UNIFAC groups as a required group is missing, it is still possible to estimate the component properties and to use this component for simulations. The number of groups from both columns is added to estimate the properties. But as soon as you add extra Joback groups knows that the UNIFAC definition is not representing the molecule and no UNIFAC groups are stored to the database to avoid wrong calculations results. To use this component correctely for simulations you will have to use other methods like NRTL or UNIQUAC to calculate activity coefficient models.
     

Atom definition table

Number and type of atoms are calculated automatically from UNIFAC / Joback group definition.
  1. Atom
    Name of Atom
     
  2. Count
    Number of Atoms in component. Atom name and count are used to calculate formula on identification tab.
     
  3. Atom
    Name of Atom
     
  4. Count
    Number of Atoms in component. Similar to the possibility to define a component by additional Joback groups it might be necessary to create a component where even no Joback group is available. Here you can enter the number of arbitrary elements for definition. But as soon as you enter any additional atomes here any available property calculation routine will collapse. You need to supply all required properties from other sources.
     
  5. Chemeo Database
    Link to Chemeo database. Here you can find Joback group definitions and publicaly available component data of many components.
