Triangular Phase Envelope - LLE

The Triangular Phase Envelope utility allows the visualisation of liquid-liquid equilibria phase maps between three different components.

The blue area depicts the compositions where the mixture is unstable and splits into two liquids. The konode lines inside that area connect two points (yellow) which represent the two liquid compositions in equilibrium with each other. Edges of that diagram represent a pure component whereas the border lines opposite to these edges represent compositions containing zero concentration of that component. The scale of the axis is displayed as molar fraction.

Components

Select three different components. The index of components defines the order on the diagram.

Conditions

Define conditions of temperature and pressure at which the diagram is to be calculated.

Property Package

Select Property package which is to be used for calculation.

Calculate Diagram

To calculate the diagram press button.

Data

This tab contains a table with calculated equilibrium compositions (=yellow points) as molar fractions.
