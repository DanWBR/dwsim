Splitter

The splitter is a mass balance unit operation. It divides a material stream into two or three other streams

Input Parameters

Operation ModeParameter
SplitRatios:This mode allows to define the mass flow fractions to be passed to each outlet stream in the splitter. Each fraction must have a value between 0 and 1 and the total sum is automatically calculated as 1 by adjusting the outlet fraction of the stream with highest number.
StreamMassFlowSpec:This mode allows to define the absolute mass flow of each outlet stream. The last outlet stream with highest index number recives the residual mass flow to maintain the massbalance.
StreamMoleFlowSpec:This mode works as the prior one except that the mole flow is specified here.

Output Parameters

There are no output parameters for this object.
